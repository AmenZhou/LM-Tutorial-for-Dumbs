🐢 Python for-loop
🚀 NumPy vectorized

Why It Matters

🔹 A for-loop executes Python bytecode per element — interpreter overhead per step.
🔹 NumPy delegates to pre-compiled C/Fortran routines that process the entire array in one hardware call.
🔹 In ML training, you sum gradients across millions of examples — loops would take hours; vectorization takes seconds.

📐 Matrix Dimension Matcher

Matrix multiplication A × B is only valid when A's columns = B's rows. The result shape is (A's rows) × (B's columns). Change the dimensions and see instantly if it works.

The Rule

For A(m×n) × B(n×p): the inner dimensions (n) must match. The result is m×p.

In neural networks, each layer is a matrix multiply — the shapes must "flow" from input → hidden → output. Getting this wrong is one of the most common bugs in ML code.
